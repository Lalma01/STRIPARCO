# STRIPARCO

[Magyar leírás lejjebb](#magyar)

**STRIPARCO** is a Windows content filter and screen-time guard. It blocks adult and
NSFW-AI websites locally (mainstream AI assistants stay accessible), monitors browser
titles, and lets you set a daily screen-time limit. The interface follows your system
light/dark theme and is available in English and Hungarian.

- Source code, build script and installer live in the [`Windows/`](Windows/) folder.
- Full documentation: [`Windows/DOCUMENTATION.md`](Windows/DOCUMENTATION.md).
- Code analysis / known bugs: [`Windows/BUGS.md`](Windows/BUGS.md).
- **Android port** (full feature parity): [`Android/`](Android/) — see [`Android/README.md`](Android/README.md).
- All filtering happens on the device — no data leaves the machine.

## Magyar

A **STRIPARCO** egy Windows tartalomszűrő és képernyőidő-felügyelő alkalmazás. Helyben
tiltja a felnőtt és NSFW-AI oldalakat (a bevett AI asszisztensek elérhetők maradnak),
figyeli a böngészők címsorát, és napi képernyőidő-korlát állítható be vele. A felület a
rendszer világos/sötét témáját követi, és magyar, illetve angol nyelven is használható.

- A forráskód, a build szkript és a telepítő a [`Windows/`](Windows/) mappában található.
- Részletes dokumentáció: [`Windows/DOCUMENTATION.md`](Windows/DOCUMENTATION.md).
- Minden szűrés a gépen történik — semmilyen adat nem hagyja el a számítógépet.
